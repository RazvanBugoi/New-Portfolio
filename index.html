Projects I'm proud of

Tic-Tac-Toe

Tic Tac Toe is a simple application that emulates the functionality of the Tic-Tac-Toe paper-and-pencil game. You can check out more about the game and it's history on wikipedia. The objective of this game was to be able to practice factory functions in JavaScript. You can have a try at the game here or check out the code on Github.

Technologies used include:

  • HTML
  • CSS
  • JavaScript

Library

Library is a project that allow users to create their own online library. Once a book has been added to the library, it's status can be changed from 'read' to 'unread' and back as needed. Another functionality is the possibility to delete a book from the library. The aim of this project was to get myself comfortable using the object constructor in JavaScript. You can try the application here or check out the code on Github.

Technologies used include:

  • HTML
  • CSS
  • JavaScript

Etch-A-Sketch

Etch-A-Sketch is a simple drawing board application. Users can select the grid size of the drawing board as well as the drawing colors. The aim of this project was to dinamically modify the DOM. You can have a play here or check out the code on Github.
